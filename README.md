# Cinematic Apology Experience

A premium, interactive web experience designed for reflection, sincere apology, and respect. 

## Features
- **Cinematic Transitions**: GSAP-powered scene management.
- **Interactive Elements**: Animated envelope, floating quotes, and masonry gallery.
- **Performance Optimized**: Uses minimal footprint, vanilla JS, and hardware-accelerated CSS.

## Assets Setup
To ensure the project runs correctly, place your files in the following directory structure:

/assets
  /images
    - background.jpg (Aurora effect)
    - memory1.jpg, memory2.jpg... (Gallery items)
  /music
    - ambient.mp3 (Soft, relaxing instrumental)
  /icons
    - favicon.ico
  /fonts
    - (Include locally if not using CDN)
